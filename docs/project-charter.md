# Паспорт проекта

## Название

Document Intake OS

## Product owner

Виталий.

## Цель

Сократить ручную подготовку документов водителей и транспорта для трех терминальных заявок при полном локальном хранении и обязательном контроле оператора.

## Результат MVP

Устанавливаемая программа для Windows 11, которая:

- импортирует исходные фотографии;
- сохраняет оригиналы без изменений;
- помогает разделять, обрезать, выравнивать и склеивать документы;
- выполняет локальное OCR/MRZ;
- позволяет проверить поля;
- ведет локальную базу;
- создает JPEG до 1,90 МиБ;
- формирует три варианта Excel;
- создает папку заявки для ручной загрузки.

## Ограничения

- offline after installation;
- real PII stays local;
- Windows first;
- operator-in-the-loop;
- three real templates;
- no direct Konversta integration;
- no real data in cloud development tools.

## Критерий успеха

Оператор может обработать типовой комплект от сырых фотографий до принятого терминалом Excel и подготовленных JPEG без повторной ручной обработки.

## Governance

- изменение бизнес-правила — явное решение;
- изменение безопасности — ADR;
- изменение Excel-адаптера — golden test;
- переход этапа — приемка gate;
- один PR — одна задача.

## Оценка

Рабочий Windows MVP: ориентир 8–12 недель после закрытия блокирующих вопросов.

## Допущения

- шаблоны будут очищены от реальных строк;
- Microsoft Excel доступен, если `.xls` обязателен;
- пилот проходит на Windows 11;
- реальные документы используются только локально.
