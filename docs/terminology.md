# Терминология

| Термин | Определение |
|---|---|
| Партия загрузки | Набор файлов одного рабочего сеанса. |
| SourceFile | Неизменяемая локальная копия входного файла. |
| Область документа | Полигон документа/стороны на исходном фото. |
| Логический документ | Один паспорт, ID, карта или транспортный документ из одного/нескольких фото. |
| Оригинал | Байты файла при импорте. |
| Рабочее изображение | Декодированное изображение для обработки. |
| Подготовленный файл | JPEG RGB ≤1,90 МиБ для сайта. |
| Recipe | Параметры преобразования. |
| OCR | Локальное распознавание текста. |
| MRZ | Машиночитаемая зона паспорта/ID. |
| FieldCandidate | Предложенное значение с источником и confidence. |
| VerifiedField | Значение, подтвержденное оператором. |
| Критическое поле | Поле, блокирующее export без проверки. |
| Conflict | Несогласованность источников или правил. |
| Person | Карточка водителя/посетителя. |
| Vehicle | Тягач, прицеп, полуприцеп или иное ТС. |
| Assignment | Связь человека и транспорта в заявке. |
| Terminal adapter | Правила заполнения одного шаблона. |
| Template manifest | Версия, checksum и структура template. |
| Golden file | Эталонный Excel на synthetic data. |
| Заявка | Изменяемый рабочий набор до snapshot. |
| Snapshot | Неизменяемые данные export. |
| ExportRun | Одна попытка формирования комплекта. |
| Manifest | Реестр файлов, checksums and versions. |
| Офлайн | Отсутствие необходимости и попыток network access. |
| PII | Персональные данные и идентификаторы. |
| Конверста | Внешняя система ручной подачи; не интегрируется в MVP. |
